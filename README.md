# Автотесты на Selenium для проверки функциональности поиска в Википедии

Проект содержит автотесты на Selenium для проверки функциональности поиска в Википедии:
- Позитивный тест: проверка информации о содержании кислорода в атмосфере Земли (20,95%)
- Негативный тест: проверка, что в статье НЕ указано некорректное содержание кислорода (например, 100%)

## Требования

- Python 3.7 или выше
- Google Chrome браузер
- pip (менеджер пакетов Python)

## Установка

1. Установите зависимости:
```bash
pip install -r requirements.txt
```

## Структура проекта

- `conftest.py` - общие фикстуры (инициализация WebDriver)
- `test_oxygen_content.py` - позитивный тест (проверка содержания кислорода 20,95%)
- `test_incorrect_oxygen.py` - негативный тест (проверка отсутствия некорректного содержания кислорода)

## Запуск тестов

### Запуск всех тестов:
```bash
pytest -v
```

### Запуск только позитивного теста:
```bash
pytest test_oxygen_content.py -v
```

Или:
```bash
python test_oxygen_content.py
```

### Запуск только негативного теста:
```bash
pytest test_incorrect_oxygen.py -v
```

Или:
```bash
python test_incorrect_oxygen.py
```

### Запуск конкретного теста:
```bash
pytest test_oxygen_content.py::TestOxygenContent::test_oxygen_content_in_atmosphere -v
pytest test_incorrect_oxygen.py::TestIncorrectOxygenContent::test_incorrect_oxygen_content_not_present -v
```

## Описание тестов

### Позитивный тест: `test_oxygen_content_in_atmosphere`

Проверяет, что в статье Википедии о Земле указано содержание кислорода 20,95% в атмосфере.

Выполняет следующие действия:
1. Открывает главную страницу Википедии (ru.wikipedia.org)
2. Вводит в поисковую строку "Земля"
3. Переходит на статью о Земле
4. Ищет в тексте статьи информацию о содержании кислорода 20,95% в атмосфере
5. Проверяет, что информация найдена и находится в контексте, связанном с атмосферой

### Негативный тест: `test_incorrect_oxygen_content_not_present`

Проверяет, что в статье Википедии о Земле НЕ указано некорректное содержание кислорода (например, 100%) в атмосфере.

Выполняет следующие действия:
1. Открывает главную страницу Википедии (ru.wikipedia.org)
2. Вводит в поисковую строку "Земля"
3. Переходит на статью о Земле
4. Ищет в тексте статьи информацию о содержании кислорода в атмосфере
5. Проверяет, что в статье НЕ указано некорректное значение (например, 100% кислорода)
6. Убеждается, что статья содержит корректную информацию (20,95% кислорода)

## Примечания

- WebDriver для Chrome устанавливается автоматически через встроенный Selenium Manager (Selenium 4.6+)
- Общий fixture `driver` находится в `conftest.py` и используется всеми тестами
- Каждый тест запускается в отдельной сессии браузера (scope="function")
- При проблемах с ChromeDriver убедитесь, что установлена последняя версия Google Chrome

## Решение проблем

Если возникает ошибка с ChromeDriver:
1. Убедитесь, что установлена последняя версия Google Chrome
2. Попробуйте обновить зависимости: `pip install --upgrade selenium`
3. Убедитесь, что у вас установлена 64-битная версия Python (если используете 64-битный Chrome)
4. Если в PATH есть старый ChromeDriver, удалите его - Selenium Manager автоматически подберет правильную версию
